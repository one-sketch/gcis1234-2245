Homework 01

Due: Wednesday, August 28, 2024, 11:59 PM

Important to Note

Note: You're encouraged to submit your work as you complete each question, but only your final submission will be counted. Ensure all three questions are included in that submission.

Note: Only question 1 will be graded automatically by Gradescope (it can't grade turtle output, which will be graded by a human). Don’t worry if your grade seems low; question 1 is just to confirm your ability to submit properly. If you have submission issues, contact your instructor.

Task

  • (20 pts) Write a function hello that prints the text Welcome to CS1 @ RIT!

  • (40 pts) Use the Python turtle library to write a function donut() that draws two concentric circles. Specifically, your code should draw a circle with radius 50 and another with radius 100, with the same center point. Recall that the circle command draws a circle in which the turtle starts at the edge of the circle, not the center! Also recall that you will need to import the turtle library.

  • (40 pts) Write a Python function called draw_plus() that draws a + sign 100 pixels high and 100 pixels wide and returns the turtle to its original position and orientation. Then write a function called draw_asterisk() that uses draw_plus() to draw an eight-pointed asterisk ✳ .
